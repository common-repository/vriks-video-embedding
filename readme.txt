A Player Plug-in based on the video.js library to embed your videos from https://vriks.de/ and https://familienvideo.de/ into your WordPress Site.
 
== Description ==
 
Our Vriks WordPress Player is a user-friendly plugin that supports video playback on desktops and mobile devices. 
You can easily embed video files that are hosted on https://vriks.de/ and https://familienvideo.de/ using the Videojs library.

= VRIKS HLS PLAYER FEATURES =

A few notes about the sections above:
 
*	Embed videos hosted on https://vriks.de/ and https://familienvideo.de/ into a post/page or anywhere on your WordPress site
*	Embed responsive videos for a better user experience while viewing from a mobile device
*	Embed HTML5 videos which are compatible with all major browsers
*	No setup required, simply install and start embedding videos


= VRIKS PLAYER PLUG-IN USAGE =

In order to embed a video create a new post/page and use the following shortcode:


`[vriks-embed src="https://vriks.de/videos/v/186"]`


The parameter "src" is the location of the video hosted on https://vriks.de/ or https://familienvideo.de/. You can copy the URL from your browsers' address-bar while being on the site where your video is shown on one of the domains.

== Installation ==
 
1. Go to the Add New plugins screen in your WordPress Dashboard
2. Click the upload tab
3. Browse for the plugin file (wp-vriks-video-embedding.zip) on your computer
4. Click "Install Now" and then hit the activate button
